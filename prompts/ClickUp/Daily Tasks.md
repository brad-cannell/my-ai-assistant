Find all tasks, including subtasks, with a due date or ‘🗓️ Target Date’ set to today. List every task and subtask (as separate items) that are due today or have a custom field ‘🗓️ Target Date’ of today. Please ignore "Daily Planning" tasks.

## Weekly version

List every task and subtask (as separate items) that are due this week or have a custom field ‘🗓️ Target Date’ of this week. Please ignore "Daily Planning" tasks. The list you return should include the task name and task id.
