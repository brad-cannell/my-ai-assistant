I use a task management application called "ClickUp." In January 2025, ClickUp released a new calendar feature in their application. Along with the calendar, they also released a set of features that allows users to time-block the tasks they've created in ClickUp on that calendar. 

Here is a link to ClickUp's help page for their feature: https://help.clickup.com/hc/en-us/articles/29051586233879-Use-the-Calendar-to-schedule-and-join-events

Here is a link to the time-blocking feature's feedback page: https://feedback.clickup.com/feature-requests/p/time-blocking-1. 

I've been trying out the new calendar and time-blocking features, and I have some feedback that I hope will improve these features in the future.

'It appears as though the calendar comes with default work hours set between 8:00 AM and 6:00 PM.  When I enter "planning mode," I'm not able to add tasks to time blocks outside of those default work hours. I believe, although I'm not 100% certain, auto-schedule mode won't schedule tasks outside of these default hours either. Additionally, I don't see any settings options that would allow me to change these default work hours. I would love to see this option added in the near future. Even better, it would be really useful if you add the ability to make multiple different schedules and allow me to chose which one to assign to a given task similar to the way Motion functions. For reference, here is a link to Motion's documentation on creating custom schedules: https://help.usemotion.com/tasks-events-and-auto-scheduling-101/understanding-schedules. Thank you!'
