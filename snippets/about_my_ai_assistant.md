# About My AI Assistant

`my-ai-assistant` is a personal knowledge and workflow repository built around API-based interactions with LLMs like Claude and ChatGPT. At its core, the repo organizes reusable **prompt templates** (stored as Markdown files in a `prompts/` folder) and **conversation workflows** (Quarto documents that mix prose and code in a `conversations/` folder), with Python and the Chatlas package powering most API calls.

Beyond prompting, the repo has evolved into a lightweight documentation site — using Quarto to render a set of **vignettes** (borrowing the concept from R packages) that capture SOPs, guidelines, and worked examples for AI-assisted workflows. The site is published via GitHub Pages and lives at [github.com/brad-cannell/my-ai-assistant](github.com/brad-cannell/my-ai-assistant).

In short, it's a well-structured personal toolkit for reproducible, version-controlled AI workflows — sitting at the intersection of prompt engineering, literate programming, and developer productivity.
